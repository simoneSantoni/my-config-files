# Zathura Configuration

Vim-keybindings PDF viewer with Yaru theme.

## Theme

Light mode with Yaru colors:
- Background: `#D5CFCA` (light)
- Foreground: `#2C001E` (dark aubergine)
- Highlight: `#F9C440` (gold)
- Active highlight: `#E95420` (Ubuntu orange)

## Font

UbuntuSans Nerd Font, 14pt

## Recolor Mode

PDF recoloring is disabled by default. Enable for dark mode reading:
```
:set recolor true
```
