# Neovide Configuration

GUI frontend for Neovim.

## Settings

| Setting | Value |
|---------|-------|
| Font | FiraCode Nerd Font Mono, 8pt |
| Frame | Full window decorations |
| Maximized | Yes (startup) |
| Tabs | Enabled |
| VSync | Enabled |
| Theme | Auto (follows system) |

## Files

- `config.toml` - Main configuration
- `neovide.desktop` - Desktop entry file

Points to custom Neovim installation:
