# Kitty Configuration

Terminal emulator configuration with Ubuntu Yaru theme.

## Files

- `kitty.conf` - Main configuration
- `current-theme.conf` - Active color theme

## Features

- **Font**: UbuntuMono Nerd Font Mono, 14pt
- **Theme**: Custom Yaru-inspired colors (dark aubergine background)
- **Ligatures**: Disabled

## Key Settings

| Setting | Value |
|---------|-------|
| Font | UbuntuMono Nerd Font Mono |
| Font size | 14.0 |
| Background | `#300a24` |
| Foreground | `#eeeeec` |
| Selection | `#b4d5ff` |

## Theme Colors

The theme uses Ubuntu's Yaru palette:
- Background: Dark aubergine (`#300a24`)
- Accent colors: Orange, teal, green, gold
- ANSI colors mapped to Yaru equivalents
